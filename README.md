## Instalasi

1. Lakukan composer install
2. Buat database dengan nama altech_omega dengan detail user = root dan password = root
3. Jalankan php artisan migrate
4. Jalankan project dengan php artisan serve
5. untuk menjalankan unit test, jalankan php artisan test
6. Collection postman bisa dilihat pada Altech_omega.postman_collection.json
